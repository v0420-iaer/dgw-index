Regime categories

Each country is assigned to one of four regime types from its overall index score (0–100). The same cut-offs apply on maps and tables; colours vary continuously by score (with ticks every 10 points on the scale below) so countries can be compared inside a category.

Category Score range
Consolidated democracy 80–100
Electoral democracy 60–79
Transitional 36–59
No democracy 0–35

The index map’s “exact scores” view uses a continuous ramp across the same thresholds as above (amaranth → honey gold → teal → blues). Category labels and the results table use one solid colour per regime type.

Democratic memory and reparations

If there is no transition from authoritarian rule, the “Reparations” block (truth commissions, reparatory policies) is left out of the score and the “Legal” block absorbs the whole weight. Thus, country profiles show Does not apply instead of a number. This index applies that rule only to Guyana.
